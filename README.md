# FitControl

Sistema de gestão para academias desenvolvido para o Projeto Integrado Multidisciplinar (PIM).

## Tecnologias utilizadas
- ASP.NET Core
- C#
- SQL Server
- HTML
- CSS
- JavaScript

## Estrutura do Projeto

### Backend
API REST desenvolvida em arquitetura em camadas:
- Controllers
- Services
- Models
- Repositories

### Frontend
Protótipos funcionais das telas do sistema:
- Login
- Dashboard
- Alunos
- Planos
- Presença
- Treinos

## Objetivo
O sistema tem como objetivo auxiliar na gestão administrativa e operacional de academias.
